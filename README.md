## What is this mod?
This mod is the fourth iteration of an idea I've had for a while. It adds a PVP match system, with lots of features like map voting, timers, scoring, and tons of gamemodes, along with an API for easy extensibility.

It is designed to be able to work on its own, but make sure to try out some of the [addons] (WIP)! They integrate this base mod with many other mods for a cohesive experience.
